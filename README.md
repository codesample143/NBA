## Technologies Used

The project utilizes the following technologies and tools:

- **Python 3** – General-purpose programming language  
- **PySpark** – Big data processing framework for distributed computing  
- **SQL** – Structured query language for database operations  
- **Docker** – Containerization platform for reproducible environments  
- **VS Code** – Code editor with Remote Containers integration  
- **Git** – Version control  
- **Pandas** – Data manipulation and analysis  
- **NumPy** – Numerical computing  
- **Matplotlib / Seaborn** – Data visualization  
- **Jupyter Notebook** – Interactive Python notebooks  
- **Linux (Ubuntu)** – Base OS for containerized environments
